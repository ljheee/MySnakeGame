一、工程目录下：file.dat 是记录玩家分数排行榜的数据文件。

二、亮点：可保存游戏进度。装载进度后，可以继续游戏。查看玩家分数表。

三、bug:
得分不为0时，死亡异常。
游戏结束，记录玩家名、保存排行榜 --有时异常。

源码学习笔记：
	1、玩家分数显示，使用JDialog和JTable.
	2、保存、装载游戏进度使用FileDialog,更简单。
	3、保存游戏进度：就是把游戏参数封装成一个个类，一个个把对象写入文件。
